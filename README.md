## 🎯 Goal

This repository is built for developers who want to:

- Prepare for software engineering interviews
- Strengthen system design fundamentals
- Learn modern architectures (AI, ML, Mobile)
- Improve coding and problem solving skills
- Build a strong developer profile

---

## 🚀 How to Use

1. Start with **Coding Interview Patterns** for problem-solving
2. Move to **System Design** resources
3. Explore **AI/ML or Mobile System Design** based on your goal
4. Use **Rust books** for deep programming knowledge (optional but powerful)
5. Improve your profile using the **Resume guide**
